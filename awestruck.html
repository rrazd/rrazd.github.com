AweStruck iOS Mobile Application UI & UX

Background

The client wanted a design of a mobile application for iOS that allows users to find food trucks near them, view a menu, select items for purchase, and checkout. A well-designed solution for mobile checkout hasn't been created yet for the food truck industry.

Competitive Analysis and Research

Competitive Analysis was performed to see how other company's are tackling mobile checkout. Some competitors examined were Square Cash or PayPal mobile.
Since this is a iPhone application the iOS Human Interface Guidelines were reviewed before beginning any design work.

User Stories and User Flow

User stories were used to define requirements and ensured that the user was always in the center of design and development process:

  • As a user, I want to view food trucks near me on a map

  • As a user, I want to select a truck, view details, and see a menu

  • As a user, I want to create an account and view recent transactions

  • As a user, I want to choose items to purchase and pay for them using my phone

  • As a user, I want to rate and review a food truck, and view other ratings and reviews

Marvel prototypes were created throughout the design phase to test the UI and UX based on feedback from potential users.

Conclusion

The final product given to the client was complete wireframes and designs of an iOS mobile application that allows users to find food trucks near them, view a menu, select items for purchase, checkout and review previous purchases.
